# Melody Masters Online Store

A modern, responsive e-commerce web application for a music instrument shop.

## Features

- Product browsing without login
- Secure user authentication
- Shopping cart and checkout system
- Physical & digital product handling
- Free shipping for orders over £100
- Role-based access (Admin, Staff, Customer)
- Admin dashboard for inventory & order management

## Tech Stack

- PHP
- MySQL
- HTML5 / CSS3
- JavaScript
- Apache (XAMPP)

## Project Structure

Organized using a clean, scalable folder structure suitable for academic and real-world projects.

## Setup Instructions

1. Clone the repository
2. Move project to `htdocs`
3. Import database from `/database`
4. Run using `localhost`
